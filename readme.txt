== Description ==

Allows the creation of redirections to internal pages and posts, and external addresses.

== Installation ==

This section describes how to install the plugin and get it working.

= Installing the Plugin =

*(using the Wordpress Admin Console)*

1. From your dashboard, click on "Plugins" in the left sidebar
1. Add a new plugin
1. Search for "ezRedirect"
1. Install "ezRedirect"
1. Once Installed, if you want to manually change your domain, go to Settings > ezRedirect
1. Manage your redirections

*(manually via FTP)*

1. Delete any existing 'ezredirect' folder from the '/wp-content/plugins/' directory
1. Upload the 'ezredirect' folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Once Installed, if you want to manually change your domain, go to Settings > ezRedirect
1. Manage your redirections
